# Тест-кейсы для Automation Test Store (https://automationteststore.com/)

## 1. Фильтрация в категориях

### TC-1.1: Сортировка товаров в категории по имени (A-Z)
**Предусловие:** Открыта главная страница сайта https://automationteststore.com/

| Шаг | Действие                                                              | Ожидаемый результат                                       |
|-----|-----------------------------------------------------------------------|-----------------------------------------------------------|
| 1   | В меню с названиями категорий навести курсор на категорию "Fragrance" | Появляется выпадающее меню с подкатегориями               |
| 2   | Кликнуть по подкатегории "Men"                                        | Открыта страница категории "Fragrance > Men".             |
| 3   | В выпадающем списке "Sort By" выбрать значение "Name A - Z"           | Страница перезагружается, товары отсортированы            |
| 4   | Проверить названия всех отображаемых товаров                          | Названия товаров расположены в алфавитном порядке (A - Z) |

## 2. Проверка поисковой выдачи и корзины

### TC-2.1: Поиск товара "shirt" и сортировка результатов
**Предусловие:** Открыта главная страница сайта https://automationteststore.com/

| Шаг | Действие                                                          | Ожидаемый результат                                                                           |
|-----|-------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| 1   | В поле поиска ввести строку "shirt"                               | Поле поиска отображает введённое текст "shirt"                                                |
| 2   | Нажать кнопку поиска или клавишу Enter                            | Открыта страница результатов поиска. Отображаются товары, содержащие в названии слово "shirt" |
| 3   | В выпадающем списке "Sort By" выбрать сортировку по имени (A - Z) | Страница перезагружается, товары отсортированы                                                |
| 4   | Проверить названия всех отображаемых товаров                      | Названия товаров расположены в алфавитном порядке (A - Z)                                     |


---

### TC-2.2: Добавление 2-го и 3-го товара из поисковой выдачи в корзину со случайным количеством
**Предусловие:** Выполнен TC-2.1. Открыта страница результатов поиска по запросу "shirt", товары отсортированы по имени

| Шаг | Действие                                             | Ожидаемый результат                                                     |
|-----|------------------------------------------------------|-------------------------------------------------------------------------|
| 1   | Кликнуть по 2-му товару для перехода на его страницу | Открыта страница товара                                                 |
| 2   | В поле "Quantity" ввести случайное число от 1 до 10  | Поле количества отображает введённое число                              |
| 3   | Нажать кнопку "Add to Cart"                          | Товар добавлен в корзину. Появляется уведомление об успешном добавлении |
| 4   | Вернуться на страницу результатов поиска             | Открыта страница результатов поиска "shirt"                             |
| 5   | Кликнуть по 3-му товару для перехода на его страницу | Открыта страница товара                                                 |
| 6   | В поле "Qty:" ввести случайное число от 1 до 10      | Поле количества отображает введённое число                              |
| 7   | Нажать кнопку "Add to Cart"                          | Товар добавлен в корзину. Появляется уведомление об успешном добавлении |


---

### TC-2.3: Удвоение количества самого дешёвого товара в корзине и проверка итоговой суммы
**Предусловие:** Открыта страница корзины, в корзине находятся два товара, добавленные в TC-2.2 (2-й и 3-й из выдачи "shirt" со случайными количествами).

| Шаг | Действие                                                                    | Ожидаемый результат                                  |
|-----|-----------------------------------------------------------------------------|------------------------------------------------------|
| 1   | В таблице товаров найти строку с наименьшей ценой за единицу `lowestPrice`  | Определён самый дешёвый товар                        |
| 3   | Запомнить текущую итоговую сумму корзины `oldTotal`                         |                                                      |
| 4   | В строке самого дешёвого товара изменить количество на удвоенное: `qty * 2` | Поле "Quantity" обновлено                            |
| 5   | Нажать кнопку "Update" (или дождаться автоматического обновления)           | Корзина обновлена. Количество товара увеличено вдвое |
| 6   | Проверить новую итоговую сумму корзины                                      | Новая сумма = `oldTotal` + (`lowestPrice` * `qty`)   |

## 3. Проверка корзины

### TC-3.1: Добавление 5 случайных товаров с главной страницы со случайным количеством
**Предусловие:** Открыта главная страница сайта https://automationteststore.com/

| Шаг | Действие                                            | Ожидаемый результат                                                         |
|-----|-----------------------------------------------------|-----------------------------------------------------------------------------|
| 1   | Выбрать 5 случайных товаров с главной страницы      |                                                                             |
| 2   | Для каждого из 5 выбранных товаров выполнить:       |                                                                             |
| 2.1 | Кликнуть по товару для перехода на его страницу     | Открыта страница товара                                                     |
| 2.2 | В поле "Quantity" ввести случайное число от 1 до 10 | Поле количества отображает введённое число                                  |
| 2.3 | Нажать кнопку "Add to Cart"                         | Товар добавлен в корзину                                                    |
| 2.4 | Вернуться на главную страницу                       | Открыта главная страница                                                    |
| 3   | Перейти в корзину                                   | Открыта страница "Shopping Cart". В корзине находятся 5 добавленных товаров |


---

### TC-3.2: Удаление всех чётных по порядку товаров из корзины
**Предусловие:** Открыта страница корзины, в корзине находятся 5 товаров, добавленных в TC-3.1

| Шаг | Действие                                         | Ожидаемый результат                                                                       |
|-----|--------------------------------------------------|-------------------------------------------------------------------------------------------|
| 1   | Удалить 2 товар в корзине, нажав кнопку "Remove" | 2 товар удалён из корзины. Осталось 4 товара                                              |
| 2   | Удалить 4 товар в корзине, нажав кнопку "Remove" | 4 товар удалён из корзины. Осталось 3 товара                                              |
| 3   | Проверить, что в корзине осталось 3 товара       | Количество товаров в таблице = 3                                                          |
| 4   | Проверить, что итоговую стоимость                | Итоговая стоимость равна сумме `цены за единицу товара` * `количество` для каждого товара |
